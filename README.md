# 🚀 Multi-Link Profile Manager

A full-stack web application that allows users to create a personalized profile with multiple links, similar to platforms like Linktree. Built using modern web technologies with secure authentication and dynamic routing.

---

## 🌟 Features

* 🔐 User Authentication (Auth.js)
* 👤 Personalized user profiles (`/username`)
* 🔗 Add, edit, and manage multiple links
* 📱 Fully responsive UI
* ⚡ Fast performance with Next.js (SSR)
* 🗂️ Centralized link sharing

---

## 🛠️ Tech Stack

* **Frontend:** Next.js, React.js, Tailwind CSS
* **Backend:** Next.js API Routes
* **Database:** MongoDB
* **Authentication:** Auth.js

---

## 📂 Project Structure

```
/app
  /api        # API routes
  /[username] # Dynamic user profiles
/components   # Reusable UI components
/lib          # Database & auth config
/models       # MongoDB schemas
```

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/your-username/multi-link-profile-manager.git
cd multi-link-profile-manager
```

2. Install dependencies

```bash
npm install
```

3. Create `.env.local` file and add:

```env
MONGODB_URI
NEXTAUTH_SECRET
```

4. Run the development server

```bash
npm run dev
```

5. Open in browser

```
http://localhost:3000
```
